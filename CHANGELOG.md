# Changelog — Atlantic to Kindle

## [1.0.0] — 2026-03-28

### Přidáno
- Chrome Extension (Manifest V3) — popup UI, content-script pro extrakci článků
- Backend server (Express) — endpoint POST /process
- Generování EPUB z článku (epub-gen-memory) — preferovaný formát pro Kindle
- Generování PDF z článku (Puppeteer) — archivní kopie s Atlantic stylingem
- Odesílání emailů (Nodemailer) — na Kindle + osobní email
- Kompletní specifikace projektu
- README s instrukcemi pro spuštění
